 # Brandon Greenwell's Scoop Bucket

 Personal Scoop bucket for Windows package installations.

 ## Installation

 ```bash
 scoop bucket add bgreenwell/scoop-bucket
 ```

 ## Updating the Bucket

 To update the bucket with the latest versions of the packages, run:

 ```bash
 scoop update bgreenwell/scoop-bucket
 scoop status
 scoop update *
 ```

 ## Contributing

 If you'd like to contribute to this bucket, please fork the repository and submit a pull request with your changes. Make sure to follow the existing structure and naming conventions for packages.
